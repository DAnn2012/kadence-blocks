Custom Blocks for for Gutenberg to help extend your editing capablitlies.

== Description ==

Build better content with Gutenberg using Kadence Gutenberg blocks. 

= Custom Blocks Include =
* Row Layout.
* Icon.
* Advanced Button.
* Spacer / Divider.

= Support =

We are happy to help as best we can with questions! Please use the support forums.

== Installation ==

Install the plugin into the `/wp-content/plugins/` folder, and activate it.
